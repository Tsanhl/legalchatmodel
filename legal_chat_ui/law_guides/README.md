# Law Subject Guides

- These guides are privacy-safe backend prompts for course-led legal answer generation.
- Use course materials first: outlines, slides, tutorial questions, exam guidance, and anonymised feedback lessons.
- Use indexed resources second: statutes, cases, textbooks, journals, and official materials.
- Each law guide should contain law-by-law answering method, topic map, material-led emphasis, issue checklist, case brief bank, feedback rules, and avoid rules.
- Where a subject has course-bound and no-limitation modes, separate scope rules from answer-quality rules: scope may change, but thesis, authority precision, structure, and verification remain mandatory.
- Do not store raw local paths, private filenames, student names, institution identifiers, or marker wording.
- Case briefs are paraphrased and exam-use focused: facts, held, reasoning, and answer use only.
